# Lead Capture AI — Oplify Solutions Assignment

## Project Overview
AI-powered lead capture system that scores leads as Hot/Warm/Cold and generates email drafts.

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS (Vercel)
- Backend: Node.js + Express + MongoDB Atlas (Render)
- AI: OpenAI GPT-4o

## Live Links
- Frontend: [https://lead-capture-agent.vercel.app](https://lead-capture-agent.vercel.app/)
- Backend: [https://lead-capture-agent.onrender.com](https://lead-capture-agent.onrender.com/)

## Setup Instructions
1. Clone the repo
2. Add .env with MONGODB_URI and OPENAI_API_KEY
3. npm install && node server.js
